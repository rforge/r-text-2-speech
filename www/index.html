Welcome to the homepage of R project “R-text-to-speech”!

The project summary page you can here.


The main purpose of this project is to implement an R package “Rtts” which helps convert text into speech (.mp3, .flv and .wav files) in R environment.

This purpose is fulfilled using APIs offered by

(1) Google Translate (https://translate.google.com)

(2) ITRI TTS @web (http://tts.itri.org.tw)


These two APIs correspond to two main functions in this package:

(1) tts_google(): English is supported now.

(2) tts_ITRI(): English and Chinese (both traditional and simplified) are supported now. In following version, I'll add more features, like change the speaking speed and speaker gender. (Reminder)

The version 0.2 of package Rtts is already ready online. You can install it by run


install.packages("Rtts", repos="http://R-Forge.R-project.org")
